# BreadcrumbsStudent-CRUD-project-with-Thymeleaf

## How to Run

1. Run the Spring Boot application:

    ```bash
    mvn spring-boot:run
    ```

2. Access the app at:

    ```
    http://localhost:8081/api/v1/students
    ```
